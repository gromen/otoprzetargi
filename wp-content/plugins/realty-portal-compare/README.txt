An add-on that provides properties comparison to your site. Work with any theme.

== Description ==

If you are looking for a plugin that helps users easily and quickly choose the most suitable property, **Realty Portal plugin** is the right solution with **Compare Property add-on** included. 

With Comparison button displayed right in the property listing, you can quickly put it in the comparison list and make a comparison later. With this add-on, you can compare 4 properties at the same time with criteria of *Price*, *Property Type*, *Offer*, *Area*, *Basic information*, and *Amenities* of properties, etc. 
Generally, whatever information the properties displays, they will be taken under comparison. Therefore, users will have a clearer view before they make a purchase decision. The more convenience you create for your users, the more loyal they are to you. It is the truth!

**Highlight features:**
* Easily pick any property and make a comparison 
* Compare all criteria properties display
* Make a comparison anytime and anywhere
* Powerful and friendly short codes.
* The maximum is up to 4 properties.
* Work with any theme.

== Installation ==

1. Upload `realty-portal-compare` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
