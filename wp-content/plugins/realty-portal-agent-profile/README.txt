== Description ==

**Agent Profile** is an add-on working on base of **Realty Portal plugin** with mainstream function of managing all information of an agent and easily edit anything in the front end. Accessing the back-end? Forget that old way!

Activating this add-on and all will be in front of you! **Agent Profile** not only reveals package that you submitted but also easily help you move to another package if you wish.
In addition, inherited the core of Realty Portal plugin that helps **manage** all information in the **front-end**, Agent Profile add-on allows you to change your personal information without accessing the back-end, such as social accounts, User name and Password, etc.

**Highlight features:**
* Update and change package plan.
* Update personal information, such as social network accounts and Login information.
* Manage all your information in front-end
* Powerful and friendly short codes.
* Work with any theme.

1. Upload `realty-portal-agent-profile` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
