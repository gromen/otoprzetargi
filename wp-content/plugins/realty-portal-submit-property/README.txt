== Description ==

**Property submission add-on** is another one that works on base of Realty Portal plugin. It helps agents can submit properties right in front-end with all necessary information. No hassle to access the back-end is needed with this plugin. All you have to do is activate the add-on and bingo! You bring about convenience for your customers and they will thank you for that.

With the Property Submission, all you need to do is fulfill information of **Custom Fields** and **Property Features**, upload gorgeous photos to help clients have a clear view of what you offer them. Primary information is like Price, Area, Bedrooms and Bathrooms, etc. 

In addition, you donâ€™t need to spend so much time fulfilling information of location of the property, such as: City, Zip code, Neighborhood, etc. Just enter the address and the system will **automatically** locate and fulfill all other fields. Your property will be submitted within a wink!

A few words of Custom Fields and Feature & Amenities: You need to go to the **Admin area** to set up which elements you think that they are important and need to be displayed.

**Highlight features:**
* Friendly and easy to use 
* Quickly submit properties in front-end
* Automatically locate properties 
* Powerful and friendly short codes.
* Work with any theme.

1. Upload `realty-portal-submit-property` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
