Fast, Powerful, Flexible solution for real estate agents using WordPress. Built-in responsive design and works for any theme.

== Description ==

**Agent Dashboard** is a powerful add-on working on base of Realty Portal plugin developed by Nooteam. It helps to manage properties right in the front-end. Agents don’t need to access the back-end anymore but still do everything with their properties quickly. So convenient. So cool. Isn’t it?

**Highlight features:**
* Easily manage submitted properties in front-end with every applied actions.
* Easy to Add, Edit and Delete properties submitted 
* Set any property as Featured one.
* Manage expired, pending and active properties.
* Powerful and friendly short codes.
* Work with any theme.

== Installation ==

1. Upload `realty-portal-agent-dashboard` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
