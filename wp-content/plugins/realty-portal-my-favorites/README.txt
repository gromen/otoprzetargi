The add-on allows user to save your properties as favorite. Work with any theme!

== Description ==

**My Favorite add-on** create a favorite list for all users, allows you to **add** and **remove** any property at any time easily.

**Features**
* Powerful and friendly short codes.
* Easy to use and friendly
* No need to set up. Just activate the add-on and you are done!
* Work with any theme.

== Installation ==

1. Upload `realty-portal-my-favorites` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
