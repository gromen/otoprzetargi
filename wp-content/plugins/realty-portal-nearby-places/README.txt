== Description ==

**Realty Portal – Nearby Places** is a powerful add on that shows places nearby the property. It gets API from **Walkscore** and **Yelp** websites to quickly locate places near the property.
With Nearby Places add-on, you can easily display Walkscore and Yelp on from your website, and it will require API key. Don’t worry! There is always step by step guide of how to get the API key for you to follow.

* What will be presented by **Yelp**?
The add-on provides you with various terms of places that you can choose to display, such as places of Education, Food, Shopping, Travel, Active Life, Night Life, etc. You can show whatever you want in property detail page. Superb, right!
Moreover, you can totally set the limit of displayed places by configure “Result Limit” in back-end area of Realty Portal plugin.

* How about **Walkscore**?
Walkscore measures the walkability of address based on the distance to nearby places and pedestrian friendliness. The score lets you know where you can reach on foot and where you cannot. The displayed locations on map will help you to easily plan your commute schedule.
 * Clear and detailed guide to get API key
 * Quickly display places around the property
 * Easily plan the commute schedule
 * No coding skill is required
 * Easy to configure as you wish
 * Support various terms of places and Distance Unit.
 * Work with any theme
