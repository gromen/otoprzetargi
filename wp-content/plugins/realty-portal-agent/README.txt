An add-on to manage agents and their information right in the front-end.

== Description ==

**Agent add-on** is one of powerful add-ons working with Realty Portal plugin. It helps to show your users list of agents and definitely, you can show this list if you wish by activating or deactivating this add-on.

**Highlight features:**
* Register or Login function
* Effortlessly display agent listing in front end by using available short code
* Directly contact agent in each property with a click.
* Fully show detailed information of agents.
* Powerful and friendly short code
* Work with any theme.

== Installation ==

1. Upload `realty-portal-agent` folder to the `/wp-content/plugins/` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
